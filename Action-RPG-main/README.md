
# God of War & Elden Ring Inspired MMORPG Action RPG

An ambitious, next-generation MMORPG action RPG inspired by God of War and Elden Ring, built in Unreal Engine 5.4 using C++ and Blueprints. This project aims to deliver visceral combat, deep lore, and a vast, interconnected world for both solo and multiplayer experiences.

## Vision

Create a living, breathing world where players can:
- Engage in skill-based, cinematic combat reminiscent of God of War
- Explore a mysterious, interconnected open world inspired by Elden Ring
- Cooperate or compete with others in a persistent online universe
- Experience epic boss battles, meaningful progression, and rich storytelling

## Key Features

- **Cinematic, Weighty Combat:** Light, heavy, and special attacks, combos, parries, and dodges
- **Deep RPG Systems:** Stats, leveling, gear, skills, and unique character builds
- **Open World Exploration:** Non-linear world design, secrets, dungeons, and dynamic events
- **Epic Boss Fights:** Challenging, multi-phase encounters with unique mechanics
- **MMORPG Elements:** Persistent world, PvE and PvP, guilds, trading, and world events
- **Rich Lore & Storytelling:** Environmental storytelling, quests, and branching narratives
- **Next-Gen Visuals:** High-fidelity graphics, dynamic weather, and day/night cycles

## Technical Overview

- **Engine:** Unreal Engine 5.4 (C++ & Blueprints)
- **Networking:** Dedicated server architecture for seamless multiplayer
- **Animation:** Advanced animation blueprints for fluid, responsive character movement
- **Ability System:** Modular gameplay abilities for combat, magic, and traversal
- **AI:** Smart enemy AI with group tactics and boss behaviors

## Getting Started

1. Clone the repository
2. Open `Warrior.uproject` in Unreal Engine 5.4+
3. Build the project (C++ required)
4. Play in editor or package for your platform
