# Combat System

## Overview
Handles all combat interactions, including attacks, damage, defense, and status effects.

## Key Functions
- PerformAttack(Target)
- CalculateDamage(Attacker, Defender, Ability)
- ApplyStatusEffect(Target, Effect)
- BlockAttack()
- Dodge()
- Parry()
- HandleDeath()

## Combat Types
- Melee
- Ranged
- Magic
- Area-of-Effect
- Boss mechanics
