# Dialogue System

## Overview
Handles all in-game conversations, branching dialogue, and player choices with NPCs.

## Key Functions
- StartDialogue(NPC)
- DisplayDialogueLine(LineID)
- PresentChoices(ChoicesArray)
- HandlePlayerChoice(ChoiceID)
- EndDialogue()

## Dialogue Types
- Linear
- Branching
- Quest-related
- Random/ambient
