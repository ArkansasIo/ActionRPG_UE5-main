# SoulsFramework Open World Asset Structure

This folder contains modular assets for open world creation, including:
- World zones
- Landmarks
- NPCs
- Bosses
- Items
- Blueprints
- Maps
- Images (concept art, 3D renders, minimaps)

Each subfolder is ready for Unreal Engine asset import and extension.

## Example Usage
- Place 3D models, textures, and materials in the Images folder.
- Add zone/landmark DataAssets and Blueprint classes in their respective folders.
- Use Maps for level layouts and minimaps.
- Populate NPCs and Bosses with character blueprints and AI configs.
